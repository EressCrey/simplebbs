# kkbbs

基于springboot+vue的学习资源分享平台(多人论坛/博客)

又名**知享乐园**

该项目旨在用于一个专属开发者的学习资源共享平台，聊天论坛。

### 快速开始

#### 后端

##### 环境要求

需要jdk1.8 + springboot2

##### 前置要求

导入kkbbs.sql

#### 前端

##### 环境要求

node版本在16.xx

分别到客户端和管理端根目录下运行

``` bash
npm install
npm run dev
```
